# VibeNotes Pro

Приложение для заметок и напоминаний с поддержкой голосовых записей.

## Что умеет

- Текстовые задачи с напоминанием по дате и времени (разовые, ежедневные, ежегодные)
- Заметки с подробным текстом и опциональным напоминанием
- Голосовые записи через микрофон
- Звуковое оповещение при срабатывании напоминания
- Значок в системном трее — сворачивается вместо закрытия
- Автозапуск при старте Windows (только EXE-версия)
- Интерфейс на русском и английском языках

---

## Установка и запуск (Windows)

### Шаг 1 — Установи Python

Скачай с https://www.python.org/downloads/ (версия 3.8 или выше)

При установке **обязательно** поставь галочку **"Add Python to PATH"**

Проверь в PowerShell:
```
python --version
```
Должна появиться строка вида: `Python 3.11.9`

---

### Шаг 2 — Распакуй архив

Распакуй ZIP в любую удобную папку, например `C:\VibeNotes\`

---

### Шаг 3 — Открой командную строку в папке приложения

В проводнике перейди в папку с файлами.
Кликни по адресной строке, напечатай `cmd` и нажми Enter.

---

### Шаг 4 — Установи зависимости

```
pip install flask pywebview pystray pillow
```

---

### Шаг 5 — Запусти приложение

```
python launcher.py
```

Откроется окно VibeNotes Pro. Значок появится в системном трее (правый нижний угол).
Крестик сворачивает в трей. Выход — правая кнопка на значок → «Выход».

---

## Сборка EXE-файла

Чтобы получить один `.exe` без необходимости держать Python:

**Установи всё нужное:**
```
pip install flask pywebview pystray pillow pyinstaller
```

**Собери EXE (вводи как одну команду, ^ это перенос строки):**
```
pyinstaller --onefile --noconsole ^
  --add-data "templates;templates" ^
  --add-data "static;static" ^
  --hidden-import pystray._win32 ^
  --hidden-import PIL._imaging ^
  --name VibeNotes ^
  launcher.py
```

Сборка займёт 2–5 минут.
Готовый файл — в папке `dist\VibeNotes.exe`

При первом запуске EXE:
- Windows покажет предупреждение → нажми **«Подробнее»** → **«Выполнить»**
- Приложение само пропишется в автозагрузку Windows

## Запуск в браузере (запасной вариант)

Если не нужно нативное окно:

```
pip install flask
python app.py
```

Открой в браузере: http://127.0.0.1:5000

---

## Где хранятся данные

- При запуске через Python — рядом с `app.py`
- При запуске EXE — рядом с `VibeNotes.exe`

Файлы:
- `app_data.json` — все заметки и напоминания
- `voice_notes\` — голосовые записи

Создаются автоматически при первом запуске и сохраняются между перезапусками.

## Частые вопросы

**Нет окна, открылся браузер?**
Установи все зависимости: `pip install flask pywebview pystray pillow`

**Нет значка в трее?**
Нажми стрелочку `^` рядом с часами — значок может быть скрыт.

**Данные пропали?**
Запускай из той же папки, где лежат файлы приложения.
