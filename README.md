# Напоминалка - Приложение для Windows 10

Приложение для создания заметок и напоминаний с поддержкой текста и голоса.

## 📋 Функции

- **Текстовые заметки** - создание, редактирование, сохранение и удаление заметок
- **Голосовые заметки** - запись голосовых сообщений через микрофон
- **Напоминания** - установка напоминаний с датой и временем
  - Текстовые напоминания
  - Голосовые напоминания (с воспроизведением записи)
- **Звуковое оповещение** - звуковой сигнал при наступлении напоминания
- **Всплывающие окна** - мигающее окно с усиленным визуальным эффектом

## 🚀 Установка

### Шаг 1: Установите Python
Убедитесь, что у вас установлен Python 3.8 или выше:
- Скачайте с https://www.python.org/downloads/
- При установке отметьте галочку "Add Python to PATH"

### Шаг 2: Установите зависимости

Откройте командную строку (cmd) или PowerShell и выполните:

```bash
pip install pygame pyaudio numpy
```

**Примечание для PyAudio:**
Если установка PyAudio не удается, скачайте готовый wheel-файл с https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
и установите его:
```bash
pip install путь_к_файлу\PyAudio‑0.2.13‑cp3xx‑...whl
```

### Шаг 3: Запустите приложение

```bash
python reminder_app.py
```

## 📖 Использование

### Работа с заметками
1. Перейдите на вкладку "📝 Заметки"
2. Нажмите "➕ Новая" для создания новой заметки
3. Введите заголовок и текст заметки
4. Нажмите "💾 Сохранить"

### Создание напоминания
1. Перейдите на вкладку "⏰ Напоминания"
2. Нажмите "➕ Добавить напоминание"
3. Выберите тип (текстовое или голосовое)
4. Введите сообщение или выберите аудиофайл
5. Укажите дату и время
6. Нажмите "Сохранить"

### Голосовые заметки
1. Перейдите на вкладку "🎤 Голосовые заметки"
2. Нажмите "🔴 Начать запись"
3. Говорите в микрофон
4. Нажмите "⏹️ Остановить запись"
5. Можно воспроизвести, сохранить как заметку или установить как напоминание

## 📁 Структура файлов

```
reminder_app.py          # Основной файл приложения
requirements.txt         # Список зависимостей
app_data.json           # Автоматически создаваемый файл с данными
voice_notes/            # Папка для голосовых заметок (создается автоматически)
alert_sound.wav         # Файл звукового сигнала (создается автоматически)
```

## ⚙️ Системные требования

- Windows 10 или выше
- Python 3.8+
- Микрофон (для голосовых заметок)
- Динамики/наушники (для звуковых оповещений)

## 🔔 Как работают напоминания

Приложение проверяет активные напоминания каждые 5 секунд. Когда наступает время напоминания:
1. Открывается всплывающее окно поверх всех окон
2. Окно мигает желтым и красным цветом
3. Воспроизводится звуковой сигнал
4. Для голосовых напоминаний - воспроизводится аудиозапись
5. Нажмите "✓ Понятно" чтобы закрыть уведомление

## 💡 Советы

- Приложение автоматически сохраняет все данные при закрытии
- Данные хранятся в файле `app_data.json`
- Голосовые заметки сохраняются в формате WAV в папке `voice_notes`
- Для работы фонового режима держите приложение запущенным

## 🐛 Решение проблем

**Не работает запись голоса:**
- Проверьте, что микрофон подключен и выбран устройством записи по умолчанию
- Предоставьте приложению доступ к микрофону в настройках Windows

**Не слышно звук напоминания:**
- Проверьте уровень громкости системы
- Убедитесь, что динамики/наушники подключены

**Приложение не запускается:**
- Убедитесь, что все зависимости установлены
- Проверьте версию Python (должна быть 3.8+)
