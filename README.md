# VibeNotes Pro

Приложение для заметок и напоминаний с поддержкой голосовых записей.

## Что умеет

- Текстовые задачи с напоминанием по дате и времени (разовые, ежедневные, ежегодные)
- Заметки с подробным текстом и опциональным напоминанием
- Голосовые записи через микрофон (запись прямо в приложении)
- Звуковое оповещение при срабатывании напоминания
- Интерфейс на русском и английском языках

---

## Установка (десктоп, Windows)

### Шаг 1 — Установи Python

Скачай с https://www.python.org/downloads/ (версия 3.8 или выше)

При установке **обязательно** поставь галочку **"Add Python to PATH"**

Проверь после установки — открой PowerShell и введи:
```
python --version
```
Должна появиться версия, например `Python 3.11.9`

---

### Шаг 2 — Распакуй архив

Распакуй скачанный ZIP в любую удобную папку, например `C:\VibeNotes\`

---

### Шаг 3 — Открой командную строку в папке приложения

В проводнике перейди в папку с приложением.
Кликни по адресной строке, напечатай `cmd` и нажми Enter.

---

### Шаг 4 — Установи зависимости

Введи одну команду:

```
pip install flask pywebview
```

Подожди, пока всё установится (займёт 1–2 минуты).

---

### Шаг 5 — Запусти приложение

```
python launcher.py
```

Откроется окно приложения VibeNotes Pro.

---

## Запуск в браузере (альтернатива)

Если не хочешь устанавливать pywebview:

```
pip install flask
python app.py
```

Затем открой в браузере: http://127.0.0.1:5000

---

## Сборка в EXE-файл (необязательно)

Если хочешь иметь один `.exe` без Python:

```
pip install flask pywebview pyinstaller
pyinstaller --onefile --noconsole --add-data "templates;templates" --add-data "static;static" --name VibeNotes launcher.py
```

Готовый `VibeNotes.exe` появится в папке `dist\`

---

## Где хранятся данные

- `app_data.json` — все заметки и напоминания (создаётся автоматически)
- `voice_notes\` — голосовые записи (создаётся автоматически)

Оба файла находятся рядом с `app.py` и сохраняются между перезапусками.

## Требования

- Микрофон (для голосовых заметок)

## Частые вопросы

**Не открывается окно приложения?**
Убедись, что установлен `pywebview`: `pip install pywebview`

**Приложение открывается в браузере вместо окна?**
Это нормальный запасной вариант. Для нативного окна нужен pywebview.

**Данные пропали после перезапуска?**
Запускай приложение командой `python launcher.py` из папки с файлами, а не из другого места.
